`CoreUIKit` provides functional and easy to use extensions on top of `UIKit`.

## Installation

### Cocoapods
```
pod 'CoreUIKit', '0.1.0'
```

### Swift package manager
```
dependencies: [
    .package(url: "https://github.com/0vertex/CoreUIKit.git", .upToNextMajor(from: "0.1.0"))
]
```

### Carthage
```
github "CoreUIKit" ~> 0.1.0
```
